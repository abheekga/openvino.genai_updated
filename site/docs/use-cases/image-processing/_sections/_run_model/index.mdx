## Run Model Using OpenVINO GenAI

OpenVINO GenAI introduces the `VLMPipeline` pipeline for inference of multimodal text-generation Vision Language Models (VLMs).
With a text prompt and an image as input, `VLMPipeline` can generate text using models such as LLava or MiniCPM-V.

See all supported [visual language models](/docs/supported-models/#visual-language-models-vlms).
